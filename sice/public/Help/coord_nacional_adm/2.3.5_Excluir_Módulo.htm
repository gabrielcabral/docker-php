2.3.5 Excluir Módulo

Esta funcionalidade está disponível  para os perfis: Coordenador Nacional Administrador.

 

Após entrar no sistema, na tela principal, selecione no menu Manutenção a opção Módulo.

 

O sistema apresenta a seguinte tela:

 

 

Informe os parâmetros de filtro e clique no botão Confirmar.

 

O sistema apresenta o resultado de filtro, conforme tela abaixo:

 

 

 

Para cada registro recuperado é apresentado na coluna 'Ação' a opção   Excluir.

 

Selecione o módulo que deseja excluir e clique no ícone   Excluir.

 

O sistema apresenta a seguinte mensagem: Deseja realmente excluir o registro?

 

Clique no botão Confirmar para confirmar a operação.

 

O sistema exclui o módulo e apresenta a seguinte mensagem: 'Registro excluído com sucesso'.

 

O módulo não poderá ser excluído se houver curso vinculado a ele.
