3.2.4 Finalizar Turma

Esta funcionalidade está disponível  para o perfil: Tutor .

 

Após entrar no sistema, na tela principal, selecione no menu Secretaria a opção Avaliação Pedagógica.

 

O sistema apresenta a seguinte tela:

 

 

Informe os parâmetros de filtro e clique no botão Confirmar.

 

O sistema apresenta o resultado de filtro, conforme tela abaixo:

 

 

Selecione uma turma que esteja com a situação  Em avaliação e no campo + ações selecione a opção Finalizar turma.

 

O sistema apresenta a seguinte tela:

 

 

Verifique se todos os cursistas foram avaliados e clique no botão Confirmar.

 

 O sistema armazena os dados e  apresenta a mensagem: Turma finalizada com sucesso.

 

Importante:  A turma não poderá ser finalizada enquanto houver alunos para avaliação e quando a data atual for menor que o mínimo para conclusão. O mínimo para conclusão é a soma da quantidade mínima de dias dos módulos cadastrados para o curso da turma.
