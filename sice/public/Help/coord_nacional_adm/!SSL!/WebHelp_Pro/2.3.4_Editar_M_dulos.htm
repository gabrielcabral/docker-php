2.3.4 Editar Módulos

Esta funcionalidade está disponível  para os perfis: Coordenador Nacional Administrador.

 

Após entrar no sistema, na tela principal, selecione no menu Manutenção a opção Módulos.

 

O sistema apresenta a seguinte tela:

 

 

Informe os parâmetros de filtro e clique no botão Confirmar.

 

O sistema apresenta o resultado de filtro, conforme tela abaixo:

 

 

 

Para cada registro recuperado é apresentado na coluna 'Ação' a opção Editar .

 

Clique no ícone para editar informações sobre o módulo.

 

O sistema apresenta a seguinte mensagem: Deseja realmente alterar dados do curso?

 

Clique no botão Confirmar para confirmar a operação.

 

O sistema apresentará a tela Cadastro de Módulo, com seus respectivos campos carregados. Veja tela abaixo:

 

 

 

Faça às alterações necessárias.

 

Para visualizar o verso do certificado clique no botão Pré-visualizar Certificado.

 

O sistema abre uma nova aba no browser, no formato pdf,  no qual será apresentando o verso do certificado. Veja a tela a seguir:

 

 

Após realizar as alterações necessárias clique no botão Confirmar.

 

O sistema armazena os dados e apresenta a  mensagem: 'Ação realizada com sucesso'.

 

Regras de campo:

 

  1. Os campos obrigatórios deverão está preenchidos.

  2. A carga horária total do módulo não pode ser igual a zero.

  3. A carga horária total do módulo não pode ser maior que 100.

  4. A carga horária a distância do módulo não pode ser igual a zero.

  5. Se existe um módulo pré-requisito, informe-o ou marque o campo "Pré-requisito" como NÃO.

  6. A sigla do módulo não poderá ter cadastrado duplicado.

  7. O nome do módulo não poderá ter cadastrado duplicado.

  8. A soma do campo carga horária presencial mínima e  carga horária a distância mínima deve ser igual ao valor informado no campo carga horária total.

  9. O valor do campo Carga Horária não pode ser menor que 40.

  10. O mínimo para conclusão não pode ser igual a zero.

  11. O máximo  para conclusão não pode ser igual a zero.
