2.4.3 Visualizar Curso

Esta funcionalidade está disponível  para os perfis: Coordenador Nacional Administrador.

 

Após entrar no sistema, na tela principal, selecione no menu Manutenção a opção Curso.

 

O sistema apresenta a seguinte tela:

 

 

Informe os parâmetros de filtro e clique no botão Confirmar.

 

O sistema apresenta o resultado de filtro, conforme tela abaixo:

 

 

 

Para cada registro recuperado é apresentado na coluna 'Ação' a opção Visualizar .

 

Clique no ícone para visualizar informações sobre o curso.

 

O sistema apresenta a tela de cadastro de curso contendo informações bloqueadas e protegidas para edição.
