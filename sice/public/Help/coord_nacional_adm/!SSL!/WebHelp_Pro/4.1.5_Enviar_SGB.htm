4.1.5  Enviar SGB

Esta funcionalidade está disponível  para o perfil: Coordenador Nacional Administrador, Coordenador Estadual e Articulador .

 

Após entrar no sistema, na tela principal, selecione no menu Financeiro a opção Bolsas.

 

O sistema apresenta a seguinte tela:

 

 

Marque o campo " Li o texto e estou de acordo com o seu conteúdo" e clique no botão Confirmar.

 

O sistema apresenta a seguinte tela:

 

 

Informe os parâmetros de filtro e clique no botão Confirmar.

 

O sistema apresenta o resultado de filtro, conforme tela abaixo:

 

 

Para cada registro recuperado é apresentado na coluna 'Ações' a opção Enviar SGB .

 

Selecione o bolsista que deseja Enviar SGB e clique no ícone Enviar SGB ou  no campo + ações selecione a opção SEnviar SGB   .

 

O sistema apresentará a tela  Enviar SGB. Veja tela abaixo:

 

Para Enviar para o SGB o usuário deve selecionar os bolsistas que deseja enviar para o SGB e utilizar o botão .

 

O Sistema então apresenta a seguinte tela para que seja Selecionado o Coordenador estadual Responsável:
