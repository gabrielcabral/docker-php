2.1.3  Desativar Configuração

Esta funcionalidade está disponível  para os perfis: Coordenador Nacional Administrador.

 

Após entrar no sistema, na tela principal, selecione no menu Manutenção a opção Configuração.

 

O sistema apresenta a seguinte tela:

 

 

Informe um dos parâmetros de filtro para pesquisar uma configuração específica e clique no botão Confirmar.

 

Para pesquisar todas as configurações cadastradas clique no botão Confirmar.

 

O sistema apresenta o resultado de filtro, conforme tela apresentada:

 

 

Para cada registro recuperado é apresentado na coluna 'Ação' a opção Desativar .

 

Selecione a configuração e clique no ícone  para desativar a configuração.

 

O sistema apresenta a mensagem: Deseja realmente desativar a configuração? A operação é irreversível!

 

Clique no botão Confirmar para desativar a configuração.

 

O sistema exibe a mensagem: “Configuração desativada com sucesso.”

 

Ao desativar a configuração a situação da mesma passará de Ativo para Inativo,
