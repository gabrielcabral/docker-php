3.4.1 Parametrizar Documento

Esta funcionalidade está disponível  para o perfil: Coordenador Nacional Administrador.

 

Após entrar no sistema, na tela principal, selecione no menu Secretaria a opção Parametrizar Documento.

 

O sistema apresenta a seguinte tela:

 

 

Os campos com (*) asterisco são de preenchimento obrigatório.

 

Preencha os campos obrigatórios em todas as abas e selecione o botão Confirmar.

 

O sistema armazena os dados e apresenta a seguinte  mensagem: "Ação realizada com sucesso".

 

Regras:

 

  1. Todo campo obrigatório deve ser informado os dados solicitados.
