1.1 Liberar senha

Esta funcionalidade está disponível  para os perfis: Coordenador Nacional Administrador, Coordenador Nacional  Equipe, Coordenador Estadual, Articulador e Tutor.

 

Vale ressaltar que o perfil Coordenador Nacional Administrador libera a senha dos usuários de todas as UF's.

 

Após entrar no sistema, na tela principal, selecione no menu Acesso a opção Liberar senha.

 

O sistema apresenta a seguinte tela.

 

 

Faça a consulta escolhendo pelo menos os campos obrigatórios e clique no botão Confirmar.

.

Feito isso, o sistema apresenta a lista de usuários com situação Liberar e/ou Renovar. Veja a figura abaixo.

 

 

Importante:  

  • Renovar: São cadastros cujo usuário solicitou acesso para renovação de senha e/ou sua senha expirou no segweb. Deverá somente gerar nova senha nos dados no segweb

  • Liberar: São cadastros novos, ou seja, o usuário nunca acessou a aplicação. O mesmo deverá ser cadastrado no segweb.

 

Selecione pelo menos um usuário que deseja Liberar e/ou Renovar a senha e no botão + ações selecione a opção Liberar acesso.

 

O sistema apresenta a mensagem: Acesso autorizado com sucesso.

 

O usuário receberá um e-mail com login e senha de acesso e a situação de seu cadastro passará para Ativo.

 

Somente cursistas na situação da turma ativa poderão ser liberados e/ou renovados.
