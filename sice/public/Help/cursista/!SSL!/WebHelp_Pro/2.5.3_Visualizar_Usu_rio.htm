2.1.2 Visualizar Usuário

Esta funcionalidade está disponível  para os perfis: Coordenador Nacional Administrador, Coordenador Nacional Equipe, Coordenador Nacional Gestor, Coordenador Estadual, Articulador,Tutor e Cursista.

 

Após entrar no sistema, na tela principal, selecione no menu Manutenção a opção Usuários.

 

O sistema apresenta a seguinte tela:

 

 

Informe os parâmetros de filtro e clique no botão Confirmar.

 

O sistema apresenta o resultado de filtro, conforme tela abaixo:

 

 

Para cada registro recuperado é apresentado na coluna 'Ação' a opção Visualizar .

 

Clique no ícone para visualizar informações sobre o usuário.

 

O sistema apresenta a tela de cadastro de usuário contendo informações bloqueadas e protegidas para edição.
