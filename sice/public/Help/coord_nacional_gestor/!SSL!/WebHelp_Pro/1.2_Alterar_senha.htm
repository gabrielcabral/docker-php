1.1 Alterar senha

Esta funcionalidade está disponível  para os perfis: Coordenador Nacional Administrador, Coordenador Nacional  Equipe, Coordenador Nacional  Gestor, Coordenador Estadual, Articulador, Tutor e Cursista.

 

Após entrar no sistema, na tela principal, selecione no menu Acesso a opção Alterar senha.

 

O sistema apresenta a seguinte tela.

 

 

O usuário informa os campos obrigatórios e clica no botão Confirmar.

 

Os dados informados deverão ser os mesmos do usuário autenticado no sistema.

 

O sistema apresenta a seguinte tela:

 

 

Informe a sua senha atual, isto é, a senha atual do usuário autenticado no sistema e clique no botão Confirmar.

 

O sistema apresenta a seguinte mensagem: ' Senha alterada com sucesso. A nova senha foi encaminhada para o e-mail: tal.tal@tal.com.br.'
