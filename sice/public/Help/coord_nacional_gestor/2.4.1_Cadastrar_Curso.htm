2.4.1 Cadastrar Curso

Esta funcionalidade está disponível  para os perfis: Coordenador Nacional Administrador.

 

Após entrar no sistema, na tela principal, selecione no menu Manutenção a opção Curso.

 

O sistema apresenta a seguinte tela:

 

 

No canto superior da tela clique no botão Cadastrar.

 

O sistema apresenta a tela de cadastro conforme figura a seguir:

 

 

Informe os campos obrigatórios.

 

Para inserir mais de um módulo clique no botão Adicionar.

 

O sistema habilita um novo campo Módulo. Para cada módulo adicionado será apresentado do lado do campo o ícone Excluir que este ao seu acionado o módulo será excluído.

 

 

Após informar os campos obrigatórios clique no botão Confirmar.

 

O sistema armazena os dados e apresenta a  mensagem: 'Cadastro realizado com sucesso'.

 

Regras de campo:

 

  1. 1. O curso não poderá ter a mesma sigla de um curso já cadastrado.

  2. 2. O nome do curso não poderá ter o mesmo nome de um curso já cadastrado.

  3. 3. A carga horária total do curso não pode ser maior que 100.

  4. 4. A carga horária total do curso não pode ser menor que 40.

  5. 5. A soma do total de horas do módulo deve de ser igual à carga horária total do curso.

  6. 6. A quantidade de módulo informado tem de ser igual a quantidade de módulos adicionados.

  7. 7. Os módulos adicionados devem ser diferentes.
