2.1.2 Pesquisar Configuração

Esta funcionalidade está disponível  para os perfis: Coordenador Nacional Administrador.

 

Após entrar no sistema, na tela principal, selecione no menu Manutenção a opção Configuração.

 

O sistema apresenta a seguinte tela:

 

 

Informe um dos parâmetros de filtro para pesquisar uma configuração específica e clique no botão Confirmar.

 

Para pesquisar todas as configurações cadastradas clique no botão Confirmar.

 

O sistema apresenta o resultado de filtro, conforme tela apresentada:

 

 

Para cada registro recuperado é apresentado na coluna 'Ação' as opções Desativar  e Visualizar
