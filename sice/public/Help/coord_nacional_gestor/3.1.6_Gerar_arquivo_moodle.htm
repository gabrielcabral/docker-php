3.1.6 Gerar arquivo moodle

Esta funcionalidade está disponível  para o perfil  Tutor .

 

Após entrar no sistema, na tela principal, selecione no menu Secretaria a opção Turmas.

 

O sistema apresenta a seguinte tela:

 

 

Informe os parâmetros de filtro e clique no botão Confirmar.

 

O sistema apresenta o resultado de filtro, conforme tela abaixo:

 

 

Selecione uma turma que esteja com a situação Ativa e no campo + ações selecione a opção Gerar arquivo moodle.

 

O sistema apresenta a seguinte tela:

 

 

Na tela descrita acima o usuário poderá notificar cursistas moodle. Veja como proceder acionando o link informado anteriormente.

 

Verifique se os dados estão corretos e clique no botão Confirmar.

 

O sistema disponibiliza o arquivo para download. Veja a figura abaixo::

 

 

Clique no botão Abrir para abrir o arquivo ou clique no botão Salvar para salvar o arquivo no sua máquina local.
