2.5.4 Editar Usuário

Esta funcionalidade está disponível  para os perfis: Coordenador Nacional Administrador, Coordenador Nacional Equipe,  Coordenador Estadual, Articulador,Tutor e Cursista.

 

Após entrar no sistema, na tela principal, selecione no menu Manutenção a opção Usuários.

 

O sistema apresenta a seguinte tela:

 

 

Informe os parâmetros de filtro e clique no botão Confirmar.

 

O sistema apresenta o resultado de filtro, conforme tela abaixo:

 

 

Para cada registro recuperado é apresentado na coluna 'Ação' a opção Editar .

 

Clique no ícone para editar informações sobre o usuário.

 

O sistema apresentará a tela Cadastro de Usuário, com seus respectivos campos carregados. Veja tela abaixo:

 

 

A UF e Município de atuação ficarão bloqueados para edição se o usuário estiver turma ativa vinculado a ele.

 

Faça às alterações necessárias e clique no botão Confirmar.

 

O sistema apresenta a seguinte mensagem: Ação realizada com sucesso.

 

Regras:

 

  1. O e-mail informado tem de ser válido. Deverá conter os seguintes atributos: ao menos 1(um) caractere antes do caractere ”@“, ao menos 1(um) caractere e “.” após o @  e ao menos 1(um) caractere após o “.”

  2. O CPF informado tem de ser válido e existente na base de dados da Receita Federal.

  3. Todo campo obrigatório deve ser informado os dados solicitados.

  4. O CEP informado tem de ser válido e existente na base de dados dos Correios. Ao informar o CEP válido o sistema deverá carregar da base de dados dos Correios os seguintes campos: Endereço, Bairro, UF e Município.

  5. O número do telefone deverá possuir a seguinte máscara:  (xx) xxxx-xxxx

  6. A data de expedição da identidade tem de ser menor que a data atual.

  7. O campo confirmar nome completo tem de ser igual ao campo nome completo, sendo este carregado da base de dados da Receita Federal.

  8. As informações para pagamento de bolsa serão carregadas conforme UF e Municipio de atuação selecionados.

  9. Para usuário novos e que não tenha cadastro no SEGWEB o status deverá ser de "Liberação pendente".
