3.1.7 Matricular cursistas

Esta funcionalidade está disponível  para o perfil  Tutor .

 

Após entrar no sistema, na tela principal, selecione no menu Secretaria a opção Turmas.

 

O sistema apresenta a seguinte tela:

 

 

Informe os parâmetros de filtro e clique no botão Confirmar.

 

O sistema apresenta o resultado de filtro, conforme tela abaixo:

 

 

Selecione uma turma que esteja com a situação Pré-Turma  e no campo + ações selecione a opção Matricular Cursista.

 

O sistema apresenta a seguinte tela:

 

 

Adicione os cursistas que deseja matricular na turma. Para adicionar os cursistas na turma clique no botão Adicionar.

 

Após adicionar os cursistas na turma será apresentado na grade Cursistas matriculados todos os cursistas que foram adicionados.

 

Para cada cursista matriculado será apresentado na grade Ações o ícone   excluir .

 

Poderá remover um cursista apenas quando a turma estiver com a situação de Pré-turma ou Não autorizada, caso contrário, a opção Remover ficará desabilitada.

 

Para remover o cursista da turma clique no ícone excluir .

 

O sistema apresenta a seguinte mensagem: Tem certeza que deseja remover o cursista?

 

Clique no botão Confirmar para confirmar a operação.

 

O sistema apresenta a mensagem: Cursista removido com sucesso.

 

Para finalizar o processo de matrícula dos cursistas clique no botão Confirmar.

 

O sistema armazena os dados e  apresenta a  seguinte mensagem:  Matrícula efetuada com sucesso.

 

Regras:

 

  1. Ao matricular o cursista, a UF e a Mesorregião de atuação deste será  igual ao do tutor.

  2. Somente poderão ser matriculados usuários com perfil cursista.
