3.1.9 Solicitar Cancelamento

Esta funcionalidade está disponível  para o perfil  Tutor .

 

Após entrar no sistema, na tela principal, selecione no menu Secretaria a opção Turmas.

 

O sistema apresenta a seguinte tela:

 

 

Informe os parâmetros de filtro e clique no botão Confirmar.

 

O sistema apresenta o resultado de filtro, conforme tela abaixo:

 

 

Selecione uma turma que esteja com a situação Ativa  e no campo + ações selecione a opção Solicitar Cancelamento.

 

O sistema apresenta a seguinte tela:

 

 

Na seção Justificativa selecione o motivo do cancelamento e caso necessário faça uma observação.

 

Clique no botão Confirmar.

 

O sistema apresenta a mensagem: Deseja realmente solicitar o cancelamento desta turma?

 

Clique no botão Confirmar para confirmar a operação.

 

Feito isso o sistema armazena os dados e apresenta a mensagem: Solicitação de cancelamento realizada com sucesso.

 

A situação da turma passará de Ativa para Aguardando Cancelamento.
