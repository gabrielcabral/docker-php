4.1.2 Avaliar Bolsista

Esta funcionalidade está disponível  para o perfil: Coordenador Nacional Administrador, Coordenador Estadual e Articulador .

 

Após entrar no sistema, na tela principal, selecione no menu Financeiro a opção Bolsas.

 

O sistema apresenta a seguinte tela:

 

 

Marque o campo " Li o texto e estou de acordo com o seu conteúdo" e clique no botão Confirmar.

 

O sistema apresenta a seguinte tela:

 

 

Informe os parâmetros de filtro e clique no botão Confirmar.

 

O sistema apresenta o resultado de filtro, conforme tela abaixo:

 

 

Para cada registro recuperado é apresentado na coluna 'Ações' a opção Avaliar Bolsa .

 

Selecione o bolsista que deseja avaliar e clique no ícone Avaliar bolsa ou  no campo + ações selecione a opção Avaliar Bolsa.

 

O sistema apresentará a tela Avaliar bolsa. Veja tela abaixo:

 

Para cada registro recuperado é  possível avaliar como A: Apto, I: Inapto e N: Não avaliado. Caso seja avaliado como Inapto é necessário adicionar uma justificativa:

 

Na seção Justificativa selecione o motivo da inaptidão e caso necessário faça uma observação.

 

Clique no botão Confirmar.

 

O sistema armazena os dados e apresenta a mensagem: Justificativa inserida com sucesso.

 

Feito isso o sistema aguada a confirmação para finalizar a avaliação.

 

Para cada registro recuperado é apresentado a coluna 'Ações'  onde é possível:

 

Visualizar os Dados das turmas selecionando o ícone .

 

O sistema apresentará a tela Visualizar Turmas. Veja tela abaixo:

 

 

Avaliar turmas selecionando o ícone

 

O sistema apresentará a tela Avaliar Turmas. Veja tela abaixo:

 

 

Caso a turma não seja aprovada é necessário  informar o motivo da não aprovação.
