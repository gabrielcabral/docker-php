3.1.8 Rejeitar Cancelamento

Esta funcionalidade está disponível  para o perfil  Articulador .

 

Após entrar no sistema, na tela principal, selecione no menu Secretaria a opção Turmas.

 

O sistema apresenta a seguinte tela:

 

 

Informe os parâmetros de filtro e clique no botão Confirmar.

 

O sistema apresenta o resultado de filtro, conforme tela abaixo:

 

 

Selecione uma turma que esteja com a situação Aguardando Cancelamento  e no campo + ações selecione a opção Cancelar Turma.

 

O sistema apresenta a tela cancelar turma contendo informações bloqueadas e protegidas para edição. Veja a figura abaixo:

 

 

Clique no botão Rejeitar Cancelamento.

 

O sistema apresenta a  mensagem: Deseja realmente rejeitar o cancelamento desta turma?

 

Clique no botão Confirmar para proceder com a operação.

 

O sistema armazena os dados e  apresenta a mensagem: Cancelamento rejeitado com sucesso.

 

Uma turma na qual o cancelamento foi rejeitado retornará para a situação de ativo.
