3.4.3 Verificar Pendências

Esta funcionalidade está disponível  para o perfil: Coordenador Nacional Administrador, Coordenador Estadual e Articulador .

 

Após entrar no sistema, na tela principal, selecione no menu Financeiro a opção Bolsas.

 

O sistema apresenta a seguinte tela:

 

 

Marque o campo " Li o texto e estou de acordo com o seu conteúdo" e clique no botão Confirmar.

 

O sistema apresenta a seguinte tela:

 

 

Informe os parâmetros de filtro e clique no botão Confirmar.

 

O sistema apresenta o resultado de filtro, conforme tela abaixo:

 

 

Para cada registro recuperado é apresentado na coluna 'Ações' a opção Verificar Pendências .

 

Selecione a bolsa que esteja com a situação Com pendência ou Encaminhada p/ SGB e clique no ícone Verificar Pendências ou  no campo + ações e selecione a opção Verificar Pendências.

 

O sistema apresentará a tela Verificar Pendências. Veja tela abaixo:

 

 

O + ações podem ser selecionadas as opções: Cancelar bolsa, Devolver para avaliação e Reenviar para SGB.

 

Cancelar bolsa

 

Selecione pelo menos um bolsista e a opção Cancelar bolsa no + ações acionando em seguida o botão Ok. O sistema apresenta a seguinte tela:

 

Na seção Justificativa selecione o motivo do cancelamento da bolsa e caso necessário faça uma observação.

 

Clique no botão Confirmar.

 

O sistema armazena os dados e apresenta a mensagem: Justificativa inserida com sucesso.

 

Feito isso a situação da turma passará de Com pendência para Cancelada.

 

Devolver para avaliação

 

Selecione pelo menos um bolsista e a opção Devolver para avaliação no + ações acionando em seguida o botão Ok. O sistema apresenta a seguinte tela:

 

Na seção Justificativa selecione o motivo do cancelamento da bolsa e caso necessário faça uma observação.

 

Clique no botão Confirmar.

 

O sistema armazena os dados e apresenta a mensagem: Justificativa inserida com sucesso.

 

Feito isso a situação da turma passará de Com pendência para Devolvida.

 

Reenviar para SGB

 

Selecione pelo menos um bolsista e a opção Reenviar para SGB no + ações acionando em seguida o botão Ok.

 

O sistema armazena os dados e apresenta a mensagem: Bolsa (s) reenviada (s) ao SGB com sucesso.

 

Feito isso a situação da turma passará de Com pendência para Encaminhada p/ SGB.
