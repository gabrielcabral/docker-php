3.1.4 Notificar Cursistas para Avaliar o Curso

Esta funcionalidade está disponível  para os perfis:  Articulador e Tutor .

 

Após entrar no sistema, na tela principal, selecione no menu Secretaria a opção Turmas.

 

O sistema apresenta a seguinte tela:

 

 

Informe os parâmetros de filtro e clique no botão Confirmar.

 

O sistema apresenta o resultado de filtro, conforme tela abaixo:

 

 

Selecione uma turma que deseja notificar os cursistas e no campo + ações selecione a opção  Notificar cursistas para aval. curso.

 

O sistema apresenta a seguinte tela:

 

 

Selecione pelo menos um  cursista que deseja notificar e clique no botão Confirmar.

 

O sistema apresenta a mensagem: Existem <Quantidade de cursistas que não foram notificados> cursistas que não foram notificados. Deseja continuar?

 

Clique no botão Confirmar para prosseguir com a operação.

 

O sistema envia o e-mail ao cursista e  apresenta a mensagem: Notificação enviada com sucesso.

 

A notificação do cursista para avaliação do curso poderá ser realizada a partir do primeiro dia da data inicio da  turma se a situação da turma estiver Ativa.
