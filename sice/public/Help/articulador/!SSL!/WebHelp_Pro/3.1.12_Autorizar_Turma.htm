3.1.5 Autorizar Turma

Esta funcionalidade está disponível  para o perfil  Articulador .

 

Após entrar no sistema, na tela principal, selecione no menu Secretaria a opção Turmas.

 

O sistema apresenta a seguinte tela:

 

 

Informe os parâmetros de filtro e clique no botão Confirmar.

 

O sistema apresenta o resultado de filtro, conforme tela abaixo:

 

 

Selecione uma turma que esteja com a situação Aguardando Autorização  e no campo + ações selecione a opção Autorizar Turma.

 

O sistema apresenta a tela autorizar turma contendo informações bloqueadas e protegidas para edição. Veja a figura abaixo:

 

 

Verifique as informações e clique no botão Confirmar.

 

O sistema apresenta a mensagem: Deseja realmente autorizar a turma?

 

Clique no botão Confirmar para  proceder com a operação.

 

O sistema armazena os dados e  apresenta a mensagem: Turma autorizada com sucesso.

 

Feito isso a situação da turma passará de Aguardando autorização para Ativa.

 

Vale ressaltar que:

    1. a quantidade de turmas autorizadas não pode ser maior ou igual à quantidade de turmas por mesorregião, informada na configuração;

    2. a turma não poderá ser autorizada se tiver com a data inicio menor que a de autorização.
