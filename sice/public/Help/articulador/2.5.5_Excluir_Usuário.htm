2.1.5 Excluir Usuário

Esta funcionalidade está disponível  para os perfis: Coordenador Nacional Administrador, Coordenador Nacional Equipe,  Coordenador Estadual, Articulador e Tutor .

 

Após entrar no sistema, na tela principal, selecione no menu Manutenção a opção Usuários.

 

O sistema apresenta a seguinte tela:

 

 

Informe os parâmetros de filtro e clique no botão Confirmar.

 

O sistema apresenta o resultado de filtro, conforme tela abaixo:

 

 

Para cada registro recuperado é apresentado na coluna 'Ação' a opção   Excluir.

 

Selecione o usuário que deseja excluir e clique no ícone   Excluir.

 

O sistema apresenta a seguinte mensagem: Deseja realmente excluir o registro?

 

Clique no botão Confirmar para confirmar a operação.

 

O sistema exclui o usuário e apresenta a seguinte mensagem: O usuário foi excluído com sucesso.

 

O usuário não poderá ser excluído se estiver vinculado a um módulo ou matrícula.
