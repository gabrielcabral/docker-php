2.1.1 Cadastrar Configuração

Esta funcionalidade está disponível  para os perfis: Coordenador Nacional Administrador.

 

Após entrar no sistema, na tela principal, selecione no menu Manutenção a opção Configuração.

 

O sistema apresenta a seguinte tela:

 

 

No canto superior da tela clique no botão Cadastrar.

 

A tela de cadastro será apresentada, conforme a seguir:

 

 

Clique na aba Dados Gerais, informe o tipo de curso e clique no botão Confirmar.

 

O sistema armazena os dados, apresenta a seguinte mensagem: Dados registrados com sucesso.

 

A aba Mesorregião ficará habilitada somente quando a aba Dados Gerais estiver sido preenchida.

 

Clique na aba Mesorregião, o sistema apresenta a seguinte tela.

 

 

Selecione a região e clique no botão Confirmar.

 

O sistema apresenta a seguinte tela.

 

 

Informe a quantidade de turmas por UF e clique no botão Confirmar.

 

O sistema apresenta a seguinte mensagem: Dados registrados com sucesso.

O cadastramento somente será finalizado após informar a quantidade de turmas de todas as UF's de todas as regiões.

 

Para verificar os municípios da mesorregião clique no link da coluna 'Qtd de Municípios'. O sistema apresenta a seguinte tela com o detalhamento dos municípios por mesorregião.
