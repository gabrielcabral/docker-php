2.2.1 Cadastrar Período de Vinculação

Esta funcionalidade está disponível  para os perfis: Coordenador Nacional Administrador.

 

Após entrar no sistema, na tela principal, selecione no menu Manutenção a opção Período de Vinculação.

 

O sistema apresenta a seguinte tela:

 

 

No canto superior da tela clique no botão Cadastrar.

 

O sistema apresenta a tela de cadastro conforme figura a seguir:

 

 

Informe o exercício que deseja cadastrar  e clique no botão Confirmar.

 

O sistema armazena os dados e apresenta a seguinte mensagem: 'Cadastro realizado com sucesso'.
