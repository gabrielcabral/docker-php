2.2.3 Excluir Período de Vinculação

Esta funcionalidade está disponível  para os perfis: Coordenador Nacional Administrador.

 

Após entrar no sistema, na tela principal, selecione no menu Manutenção a opção Configuração.

 

O sistema apresenta a seguinte tela:

 

 

Informe o parâmetro de filtro e clique no botão Confirmar.

 

O sistema apresenta o resultado de filtro, conforme tela apresentada:

 

 

Para cada registro recuperado é apresentado na coluna 'Ação' a opção   Excluir.

 

Selecione o período de vinculação que deseja excluir e clique no ícone   Excluir.

 

O sistema apresenta a seguinte mensagem: Deseja realmente excluir o registro?

 

Clique no botão Confirmar para confirmar a operação.

 

O sistema exclui o período de vinculação e apresenta a seguinte mensagem: 'Registro excluído com sucesso'.

 

O período de vinculação só poderá ser excluído se não houver  bolsas vinculadas a ele.
