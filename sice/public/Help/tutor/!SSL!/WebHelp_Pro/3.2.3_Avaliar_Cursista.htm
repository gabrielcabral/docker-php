3.2.3 Avaliar Cursista

Esta funcionalidade está disponível  para o perfil: Tutor .

 

Após entrar no sistema, na tela principal, selecione no menu Secretaria a opção Avaliação Pedagógica.

 

O sistema apresenta a seguinte tela:

 

 

Informe os parâmetros de filtro e clique no botão Confirmar.

 

O sistema apresenta o resultado de filtro, conforme tela abaixo:

 

 

Selecione uma turma que esteja com a situação Ativa ou Em avaliação e no campo + ações selecione a opção Avaliar Cursista.

 

O sistema apresenta a seguinte tela:

 

 

Preencha os campos e clique no botão Confirmar.

 

Deve ser informado pelos uma nota para avaliação da turma. Uma turma que tenha pelo menos uma nota inserida deverá ficar com a situação "Em avaliação".

 

O sistema armazena os dados e  apresenta a mensagem: Avaliação realizada com sucesso.
