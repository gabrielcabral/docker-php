3.1.7.1 Matricular cursistas

Esta funcionalidade está disponível  para o perfil  Tutor .

 

Após entrar no sistema, na tela principal, selecione no menu Secretaria a opção Turmas.

 

O sistema apresenta a seguinte tela:

 

 

Informe os parâmetros de filtro e clique no botão Confirmar.

 

O sistema apresenta o resultado de filtro, conforme tela abaixo:

 

 

Selecione uma turma que esteja com a situação Pré-Turma  e no campo + ações selecione a opção Matricular Cursista.

 

O sistema apresenta a seguinte tela:

 

 

Para realizar o cadastro dos cursistas na turma clique no botão Adicionar.

 

O sistema apresenta a seguinte tela:

 

 

Os campos com (*) asterisco são de preenchimento obrigatório.

 

Preencha os campos obrigatórios e clique no botão Confirmar.

 

O sistema armazena os dados e apresenta a seguinte  mensagem: Cursista adicionado com sucesso e retorna para a tela Matricular cursista apresentando na seção Cursistas matriculados o cursista que foi adicionado.

 

O cursista somente ficará vinculado a turma após ser acionado o botão Confirmar na tela Matricular.

 

Regras:

 

  1. O cursista somente poderá ser adicionado se estiver cadastrado e vinculado à outra turma com as situações: Finalizada e Cancelada.

  2. O cursista se estiver com a situação de Inativo não poderá ser adicionado a nenhuma turma.
